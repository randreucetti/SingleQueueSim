To run the simulation do the following:

For SingleSimulation:

-Place all files in a directory
-Compile with "javac SingleSimulation.java"
-run with "java SingleSimulation" or "java SingleSimulation > output.txt" to output to file

For NetworkSimulation:

-Place all files in a directory
-Compile with "javac NetworkSimulation.java"
-run with "java NetworkSimulation" or "java NetworkSimulation > output.txt" to output to file
